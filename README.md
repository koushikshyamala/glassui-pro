# ✨ GlassUI Pro

An advanced, interactive Glassmorphism CSS Generator built for modern web developers and UI designers. 

### 🚀 Live Demo
**[Click here to use the Live Generator](https://koushikshyamala.github.io/glassui-pro/)**

### 🛠️ Features
*   **Visual Editor:** Instantly adjust blur, transparency, and colors.
*   **Real-time Output:** See your CSS code update automatically as you drag the sliders.
*   **Mobile Responsive:** Fully functional on desktop and mobile devices.
*   **Zero Dependencies:** Built with pure HTML, CSS, and vanilla JavaScript.

### 💻 How to Use
1. Adjust the sliders to create your perfect frosted-glass effect.
2. Click the "Copy CSS" button.
3. Paste the code directly into your project!
